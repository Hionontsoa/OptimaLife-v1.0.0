# 📖 Guide d'Utilisation - RH ZenManager v1.2

## 🎯 Vue d'Ensemble des Fonctionnalités

Bienvenue dans la version 2.0 de RH ZenManager. Ce guide vous explique comment utiliser toutes les nouvelles fonctionnalités ajoutées pour gérer vos finances, tâches et objectifs plus efficacement.

---

## 1️⃣ Gestion des Budgets

### Configurer les Budgets par Catégorie

1. Allez dans l'onglet **Finances**
2. Cliquez sur le bouton **"Configurer les budgets"** (icône roue dentée)
3. Pour chaque catégorie, entrez :
   - **Limite mensuelle** : Montant maximum à dépenser
   - **Seuil d'alerte** : Pourcentage auquel alerter (automatique à 80%)
4. Cliquez **"Enregistrer tous les budgets"**

### Comprendre les Alertes

- **Alerte Ambre** : Vous avez atteint 80% de votre budget
- **Alerte Rouge** : Vous avez dépassé 100% de votre budget
- Les alertes s'affichent automatiquement en haut de l'écran Finances

---

## 2️⃣ Onglet "Rapports" - Analyses Complètes

### Vue d'Ensemble (Default)

Affiche les statistiques globales :
- **Objectifs** : Complétés, en cours, progression moyenne
- **Productivité** : Tâches complétées vs en attente
- **Rapport du mois** : Revenus, dépenses, solde
- Sélectionnez le mois avec le sélecteur

### Tendances

Consomez les tendances de vos finances :
- **Tendances hebdomadaires** : Revenu/Dépense/Solde par semaine
- **Projections futures** : Balance estimée pour les 3 prochains mois

### Alertes

Suivi des problèmes budgétaires :
- **Budgets dépassés** : Catégories où vous avez dépassé les limites
- **Dépenses anormales** : Transactions non typiques détectées

### Export

Exportez vos données :
- **Export CSV** : Depuis Excel/Google Sheets
- **Sauvegarde JSON** : Backup complète de toutes les données

---

## 3️⃣ Tâches Récurrentes

### Créer une Tâche Récurrente

1. Ouvrez l'onglet **Tâches**
2. Entrez le titre de la tâche
3. Cliquez sur l'icône **cloche** pour le formulaire avancé
4. Configurez :
   - Date d'échéance
   - Heure de rappel
   - Priorité (Basse/Moyenne/Haute)
   - **Récurrence** : Choisissez la fréquence
   - **Intervalle** : Tous les N jours/semaines/mois
5. Cliquez **Ajouter la tâche**

### Types de Récurrence

- **Aucune** : Tâche unique
- **Chaque jour** : Se répète tous les jours
- **Chaque semaine** : Se répète hebdomadairement
- **Chaque mois** : Se répète mensuellement
- **Chaque année** : Se répète annuellement

### Indicateur de Récurrence

- Les tâches récurrentes affichent un badge **⟳ Chaque jour** (ou la fréquence)
- Couleur violet pour identifier facilement

---

## 4️⃣ Améliorations des Objectifs

### Créer un Objectif Amélioré

1. Onglet **Objectifs** → **"Créer un nouvel objectif"**
2. Remplissez :
   - **Titre** : Nom de l'objectif
   - **Montant cible** : Combien vous voulez économiser
   - **Échéance** : Date limite
   - **Icône** : Représentation visuelle
   - **Status** : Active/Complet/Abandonné

### Suivi du Statut

Les objectifs peuvent avoir les statuts suivants :
- **Active** : Objectif en cours
- **Complété** : Objectif atteint (✓)
- **Abandonné** : Objectif annulé

### Badges de Progression

- **Presque là !** (90-99%) : Badge ambre qui pulse
- **✓ Complété** (100%) : Badge vert

---

## 5️⃣ Export et Sauvegarde

### Sauvegarder Vos Données

**Automatiquement :**
- Sauvegarde automatique toutes les 5 minutes
- Visible dans Rapports → onglet Export

**Manuellement :**
1. Allez dans **Rapports** → **Export**
2. Choisissez le format :
   - **CSV** : Pour Excel/Sheets
   - **JSON** : Pour backup technique

## 8️⃣ Conseils d'Utilisation Optimale

### Stratégie de Budget

1. **Mois 1** : Suivez vos dépenses sans limite
2. **Mois 2** : Analysez les patterns
3. **Mois 3+** : Définissez des budgets réalistes

### Tâches Récurrentes

- Utilisez pour les tâches répétitives régulières
- Exemples :
  - Facture mensuelle
  - Nettoyage hebdomadaire
  - Revue mensuelle des finances

### Objectifs

- Créez des objectifs réalistes
- Définissez une échéance claire
- Contribuez régulièrement
- Suivez la progression

## 🐛 Dépannage

### Les alertes budgétaires ne s'affichent pas

- Vérifiez que vous avez configuré au moins un budget
- Assurez-vous d'avoir des transactions dans les catégories

### Les tâches récurrentes ne se créent pas

- Vérifiez que la première tâche a bien été créée
- Passez quelques secondes pour que la récurrence soit enregistrée

### Les données ne s'exportent pas

- Vérifiez que vous avez des données à exporter
- Essayez le format JSON si CSV échoue
- Vérifiez vos permissions de téléchargement
